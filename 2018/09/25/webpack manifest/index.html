一。webstrome上修改不了文件

在webstrome的项目上，发现一整个文件夹都被锁住了(一个小图标🔒)。
谷歌到sudo chown -R 你的用户名称 文件夹
例如： sudo chown -R yourname file
为什么这个能解除锁呢？
因为这个命令是 允许修改file文件权限的意思。
